# Использованные источники

- **Официальная документация humanize**  
  https://humanize.readthedocs.io/en/stable/  
  Использована для описания API: функции модулей number, time, filesize, примеры вызовов, параметры `naturalsize`, `naturaltime`, `precisedelta`, раздел по локализации (i18n.activate/deactivate).

- **Репозиторий python-humanize/humanize (GitHub)**  
  https://github.com/python-humanize/humanize  
  Использован для общей информации о проекте, списке поддерживаемых локалей, способе установки из исходников и структуре пакета.

- **PyPI — humanize**  
  https://pypi.org/project/humanize/  
  Использован для проверки актуальной версии (4.15.0), требований к Python (≥3.10) и способа установки через pip.

- **Статья «Bridge the Gap Between Data and Humanity With the Power of This Python Library» (Towards Data Science)**  
  Упоминалась в поиске как пример применения humanize в сценариях data science и визуализации для нетехнической аудитории; использована для формулировки примеров применения в реальных проектах (отчёты, интерфейсы, логи).

Локально использованы шаблоны документации из задания: структура `docs/` (overview.md, task.md, conclusions.md, sources.md) и требования к разделам модулей 1–3.
